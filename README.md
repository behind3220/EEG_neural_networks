## Download required dataset

```console
wget --load-cookies /tmp/cookies.txt "https://docs.google.com/uc?export=download&confirm=$(wget --quiet --save-cookies /tmp/cookies.txt --keep-session-cookies --no-check-certificate 'https://docs.google.com/uc?export=download&id=1njcx7s5-DQAzWfHn39Uml0t0YkfsJWgd' -O- | sed -rn 's/.*confirm=([0-9A-Za-z_]+).*/\1\n/p')&id=1njcx7s5-DQAzWfHn39Uml0t0YkfsJWgd" -O data.zip && rm -rf /tmp/cookies.txt
```

### Unzip data.zip

```console
unzip data.zip
```

## Run the model
Open main.ipynb and make sure that you can access your data.
